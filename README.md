# Nuxt 3 模板

## Package List

- Nuxt 3
- Nuxt UI 2
- Vue 3.5
- Vue Router 4
- Nuxt Color Mode 3
- Nuxt ESLint 0.6
- ESLint 9
- Prettier 3
- Sass 1
- TypeScript 5

## 設置

確保安裝依賴項：

```bash
# pnpm
pnpm install
```

## 開發服務器

在 `http://localhost:3000` 上啟動開發服務器：

```bash
# pnpm
pnpm run dev
```

## 生產環境

為生產環境構建應用程序：

```bash
# pnpm
pnpm run build
```

本地預覽生產構建：

```bash
# pnpm
pnpm run preview
```

查看[部署文檔](https://nuxt.com/docs/getting-started/deployment)以獲取更多信息。
